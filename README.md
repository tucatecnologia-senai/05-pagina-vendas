## 📋 Sobre o Projeto

Landing page de alta conversão para o **NEXUS Watch Pro**, smartwatch premium com IA integrada. O site foi projetado com foco em:

- **Performance**: HTML/CSS/JS puro, sem frameworks — carregamento instantâneo
- **Conversão**: Estrutura persuasiva com countdown, social proof e CTA estratégicos
- **Design Premium**: Paleta dourado/âmbar sobre fundo preto, animações suaves
- **SEO**: Schema.org, meta tags OpenGraph, hierarquia semântica correta
- **Acessibilidade**: Contrastes WCAG, atributos `aria`, semântica HTML5

## ⚡ Como Executar Localmente

Não há necessidade de build ou instalação. Basta abrir no navegador:

```bash
# Opção 1 — Abrir direto
# Clique duas vezes em index.html

# Opção 2 — Servidor local com VS Code (Live Server)
# Instale a extensão Live Server e clique em "Go Live"

# Opção 3 — Servidor Python
python -m http.server 3000
# Acesse: http://localhost:3000
```

## 🌐 Deploy

### Netlify Drop (Mais Rápido — Sem Conta)
1. Acesse [app.netlify.com/drop](https://app.netlify.com/drop)
2. Arraste a **pasta inteira** do projeto para a área de drop
3. Receba seu link público em segundos

### Netlify CLI
```bash
npm install -g netlify-cli
netlify deploy --dir . --prod
```

### GitHub Pages
1. Vá em **Settings → Pages** do repositório
2. Source: `Deploy from a branch`
3. Branch: `main` / pasta `/` (root)
4. Acesse: `https://tucatecnologia-senai.github.io/05-pagina-vendas`
